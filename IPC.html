Parent-Child String Case Reversal in C

A C program demonstrating IPC using pipes and process forking, where the parent sends a string to the child process which reverses the case of each letter and sends it back.

Program Overview & Communication Flow

This project showcases basic Unix process management and IPC: the parent process reads user input, sends it to the child via pipes, the child toggles letter cases, and returns the modified string.

  • Using two pipes for bidirectional communication
  • Demonstrates process forking and synchronization
  • Case conversion logic implemented in child process
  • Continuous user interaction until exit command

Though simple, this project is a clear demonstration of fundamental OS concepts and process control in C, ideal for learning low-level interprocess communication.
