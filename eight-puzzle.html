8-Puzzle Solver Using BFS in Java

A Java-based program that solves the classic 8-puzzle problem using Breadth-First Search. It reads multiple puzzles from a file and outputs the minimum steps to reach the goal state.

Solving the 8-Puzzle Problem with Java

This project loads 3×3 puzzle configurations from a text file, applies a BFS algorithm to search for the optimal sequence of moves, and reports the minimum number of steps required to solve each puzzle. If a puzzle is unsolvable, the program outputs unreachable.

  • Reads and processes up to 1000 puzzles from file
  • Efficient state representation and hash checking
  • Output includes both input and result for clarity
  • Handles solvability detection gracefully
  • Good demonstration of algorithmic thinking in Java

This solver is an academic-style implementation to practice data structures, state exploration, and algorithm design. It effectively demonstrates Java proficiency and algorithmic problem solving.

A sample output looks like: Input: 125340678 → Output: 10 steps, or unreachable for unsolvable cases.
