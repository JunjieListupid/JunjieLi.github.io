Basic HTTP Client-Server Web Page Fetcher

A lightweight client-server system built in Python where the client fetches HTML pages directly from a custom TCP server using raw HTTP GET requests.

Network Communication Using Sockets in Python

This project demonstrates a fundamental web server and client written in pure Python using the socket module. The server listens on a port and serves static HTML files, while the client connects, sends an HTTP GET request, and prints the response.

  • Manual parsing of HTTP requests and file delivery
  • Client can specify any file path to request
  • Graceful handling of 404 errors if file not found
  • Simple modular code for both server and client
  • Command-line arguments for IP, port, and file path

Though minimal in scale, this project reflects a solid understanding of how the web works under the hood — including sockets, HTTP protocol basics, and server-client data exchange — all implemented without external libraries.

It’s a useful demonstration of core networking concepts and low-level I/O operations, often hidden in high-level web frameworks.
